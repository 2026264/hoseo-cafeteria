■ 테이블 예약

앱에서 원하는 테이블을 선택하고 이름과 이용 시간을 입력하면 예약 완료. 예약 후 10분 이내 미착석 시 자동 취소됩니다.

■ 대기 신청

이용 중인 테이블 클릭 → 이름 입력 → 대기 등록. 최대 2팀까지 가능. 앞 팀 종료 시 알림과 함께 순서가 진행됩니다.

■ 실시간 현황

모든 이용자의 예약이 실시간으로 공유됩니다. 상단 초록 점(●)이 켜지면 실시간 연동 중입니다.

■ 이용 시간

30·35·40분 중 선택 가능. 시간 종료 후 대기 팀이 있을 경우 자동으로 다음 팀에게 자리가 넘어갑니다.
